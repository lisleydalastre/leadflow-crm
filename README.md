# LeadFlow CRM

CRM para gestión de leads, seguimiento comercial y medición de conversiones desarrollado con PHP, MySQL y JavaScript utilizando arquitectura MVC.

## Características

- Gestión de leads
- Seguimiento comercial
- Registro de ventas
- Dashboard con métricas
- Reportes de conversión
- Autenticación de usuarios

## Tecnologías

- PHP 8
- MySQL
- JavaScript
- HTML5
- CSS3

## Arquitectura

Patrón MVC (Model View Controller)
